🤖 AI Chatbot with Web Scraping (GeeksforGeeks Data)

This project is an AI chatbot that uses scraped data from GeeksforGeeks to generate responses based on real educational content. It includes a web scraper, dataset preparation, and a basic AI training/generation pipeline.


📌 Project Overview

The system works in three main steps:

1. Scraping Data
   Extracts AI/ML-related content from selected GeeksforGeeks pages.

2. Data Processing
   Stores scraped text in a structured format for training.

3. Chatbot Model
   Uses the processed data to train or feed an AI model for generating responses.
